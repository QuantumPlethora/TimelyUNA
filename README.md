# TimelyUNA

**A modern light-time sextant that shows you the universe as it *actually* is — not as its light claims it was.**

Because photons take time to travel (8 minutes 19 seconds from the Sun), what you see in the sky is already history. TimelyUNA corrects for that.

## Try the Live Demo
→ **[Play with the interactive HTML demo](https://QuantumPlethora.github.io/TimelyUNA)**

## What it does right now
- **Demo 1**: See the Sun’s *apparent* position vs its *actual* current position. Launch a Baby SPCX rocket that flies to the true location (correcting for light delay).
- **Demo 2**: Quantum jump 65 million light-years away, then deploy an improbable quantum telescope to look back at Earth during the age of dinosaurs.

Built as a playful proof-of-concept for a future ARKit/SwiftUI experience.
